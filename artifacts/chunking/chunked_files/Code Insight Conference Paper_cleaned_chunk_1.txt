Sapkal College of Engineering Nashik, India Abstract - The increasing demand for effective programming education highlights the need for intelligent systems that go beyond basic code evaluation. Traditional assessment platforms primarily focus on correctness and execution, often lacking meaningful feedback on code quality, efficiency, and conceptual understanding. This paper presents an educational code assessment platform that utilizes a hybrid artificial intelligence approach by combining fine-tuned and generative AI models. The proposed system integrates a fine- tuned CodeLlama 7B model for analyzing time and space complexity along with code structure, while a generative AI model (Gemini API) provides detailed explanations, optimization suggestions, and learning support. This combination enables both accurate technical evaluation and interactive educational feedback. Additionally, the platform supports automated execution, submission tracking, and a performance-based leaderboard to enhance student engagement. The hybrid approach improves the quality and interpretability of feedback compared to traditional methods, helping learners better understand programming concepts and improve code efficiency. The system demonstrates the potential of combining analytical and generative AI techniques to enhance modern programming education. INTRODUCTION The rapid expansion of the software industry and digital technologies has significantly increased the demand for proficient programming skills. As a result, programming education has become a fundamental component of academic curricula and professional training programs. However, evaluating programming assignments effectively remains a challenging task, particularly in large classrooms and online learning environments. Traditional code assessment methods and existing automated platforms primarily focus on correctness and successful execution of programs, often overlooking important aspects such as code efficiency, structure, and conceptual clarity. This limitation restricts students from receiving comprehensive feedback necessary for improving their problem-solving abilities. In recent years, artificial intelligence has emerged as a promising solution for enhancing automated code evaluation systems. Machine learning and deep learning models have been applied to analyze code patterns, detect errors, and estimate computational complexity. At the same time, generative AI models have demonstrated the ability to produce human-like explanations and assist in understanding complex programming concepts. Despite these advancements, most existing systems rely on either analytical models or generative models independently, which leads to a gap between precise technical evaluation and meaningful educational feedback. To address this challenge, this paper proposes an educational code assessment platform that employs a hybrid artificial intelligence approach. The system combines a fine- tuned CodeLlama 7B model for analyzing time complexity, space complexity, and code structure with a generative AI model (Gemini API) that provides detailed explanations, optimization suggestions, and conceptual guidance. By integrating these two approaches, the platform aims to deliver both accurate evaluation and interactive learning support within a single system. In addition to AI-based analysis, the platform incorporates features such as automated code execution, submission tracking, and a performance-based leaderboard, making it suitable for academic institutions and self-learning environments. The primary objective of this work is to bridge the gap between automated assessment and pedagogy by providing students with actionable insights that enhance their coding skills and conceptual understanding. II. LITERATURE SURVEY The evaluation of programming assignments has evolved from manual assessment to automated systems aimed at improving scalability and consistency. Early automated code assessment tools primarily focused on compiling and executing programs against predefined test cases to verify correctness. While such systems are efficient in handling large numbers of submissions, they provide limited feedback, often failing to address deeper aspects such as algorithmic efficiency, code quality, and conceptual understanding. This limitation has motivated researchers to explore more intelligent approaches for code analysis. Subsequent advancements introduced static analysis techniques and rule-based systems to evaluate code structure and detect common programming errors. These methods analyze syntax, control flow, and predefined coding standards to generate feedback. Although they improve the quality of evaluation compared to basic execution-based systems, their effectiveness is constrained by rigid rules and limited adaptability to diverse coding styles and problem- solving approaches.
